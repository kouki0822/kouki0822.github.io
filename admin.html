GitHub 設定

アクセストークンを1回設定するだけで、「GitHubに保存」が使えます。
トークンはこのブラウザだけに保存され、外部には送信されません。

未設定

トークンの発行方法：
1. github.com/settings/tokens/new を開く
2. Note（名前）に「小さいラボ」と入れる
3. Expiration（有効期限）を選ぶ
4. Scopeで repo にチェック
5. 「Generate token」→ 表示されたトークンをコピーして上に貼る
